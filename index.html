Interfaz visual interactiva de Residuos Sólidos Urbanos en CDMX

This is a 2D Processing.pde canvas web-based sketch:

Learn more

The Interactive visual interface presents a flow diagram system from the Urban Solid Waste (USW) at Mexico City (CDMX), according to the Solid Waste's inventory published by Secretaría del Medio Ambiente (SEDEMA) in CDMX in 2016. The job is been made possible in collaboration with Patricia Galán Lara.

How many solid waste flow through each station process regarding with the quantities published.

As a reference, each pipeline represents the quantity as strokeWeight from the solid waste information available in a scale divided by ten (e.g. 1,038.23 ton/día, to obtain 103.82), to take only the integer value (for now), and taste how the tools, than i'm using, works with sufficiency.
